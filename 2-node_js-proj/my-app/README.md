verify that all nodes are installed
node -v
npm -v

clone the app source repo and cd into the directory 

initialize the node.js project
npm init -y

install all dependencies needed
npm install express

cd into package.json file and add the start script as to source
"script":{
start:"node index.js"
}

run npm start

http://localhost:3000/ to your web 

stop it with ctrl^c
